# About / Contact改善計画

## Aboutに書くべき内容

### サイトの目的

AIを単なる相談相手で終わらせず、業務改善、自動化、成果物保存、再利用につなげるための実践サイトであることを明記する。

### 対象読者

- ChatGPTを使っているが業務改善につながっていない人
- CodexやClaude Codeを開発Agentとして使いたい人
- Prompt、Skill、Agentを整理して再利用したい人
- ObsidianでAI成果物を保存したい人
- 中小企業や個人事業でAI導入を進めたい人

### 扱うテーマ

- ChatGPT
- Codex
- Claude / Claude Code
- Cursor
- Gemini
- Obsidian
- Prompt
- Skill
- Agent
- MCP
- Hook
- Guardrails
- Eval
- AI秘書兼スタッフOS
- AI業務改善

### 扱わないこと

- 顧客情報や個人情報を使った実験
- 秘密情報を外部AIへ渡す運用
- 本番サーバーへの自動接続
- WordPress復旧や移行代行
- 法務、税務、医療などの専門判断の代替

### 編集方針

- 実践ログを重視する
- 手順、制約、検証方法を残す
- 成果物を再利用できる形にする
- ツール比較は役割分担を中心に書く
- 失敗条件や注意点も記録する

## Contactに書くべき内容

### 現状方針

まだフォーム処理は作らない。
外部送信、メール送信、DB保存、CRM連携は実装しない。

### 相談前に整理してほしい項目

- 改善したい業務
- 現在の作業時間
- 使っているAIツール
- 使っている保存場所
- AIに任せたい作業
- 人間が確認すべき作業
- 最終的に残したい成果物

### 将来の導線候補

- メールリンク
- Googleフォーム
- Notionフォーム
- 静的フォームサービス

## Googleフォームを使う場合の注意点

- Googleアカウント側で回答保存先を管理する
- 収集する個人情報を最小限にする
- 回答データの閲覧権限を制限する
- フォームにプライバシー説明を入れる
- スパム対策を検討する

## メールリンクを使う場合の注意点

- スパムが増える可能性がある
- メールアドレスを画像化するか、表記方法を工夫する
- 返信保証や納期保証につながる表現を避ける
- 相談対象外の内容を明記する

## 個人情報保護上の注意点

- 氏名、メールアドレス、会社名、相談内容は個人情報になり得る
- 保存場所、保存期間、利用目的を明確にする
- 外部フォームを使う場合は外部サービスの規約と保存先を確認する
- プライバシーポリシーを整備してから本格運用する
- 秘密情報や顧客情報を送らないよう注意書きを置く
