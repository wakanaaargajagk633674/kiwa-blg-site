# Kiwa Personal Media — WordPress ブロックテーマ

**Version 1.1** — UI/UX改善・内部リンク修正・さとるん表示名対応済み

screenshot.png は 1200×900 推奨です（現在未同梱）。

## 1. テーマ概要

**Kiwa Personal Media** は、40代ゲイの個人メディア向けに設計されたWordPressブロックテーマです。

- ダークネイビー × オフホワイト × くすみゴールドの落ち着いたデザイン
- Full Site Editing（FSE）完全対応
- 顔出しなしでも世界観が伝わる文字・余白中心のレイアウト
- ブログ、作品集、プロフィール、お問い合わせに展開可能
- 勤務先・個人関係者が特定されない運用設計
- WordPressが `/wp/` サブディレクトリにインストールされている場合に対応したリンク設定

---

## 2. インストール方法

### 方法A：ZIPアップロード（推奨）

1. `kiwa-personal-media` フォルダをZIP圧縮する
2. WordPress管理画面 → 外観 → テーマ → 新しいテーマを追加
3. 「テーマのアップロード」→ ZIPファイルを選択してインストール
4. 「有効化」をクリック

### 方法B：FTPアップロード

1. `kiwa-personal-media` フォルダをそのまま `/wp-content/themes/` にアップロード
2. WordPress管理画面 → 外観 → テーマ → 有効化

---

## 3. ZIP化する手順（Windows）

```
kiwa-personal-media フォルダを右クリック
→「圧縮先に送る」→「ZIPファイル」
→ できた kiwa-personal-media.zip をアップロード
```

**注意：** フォルダごと圧縮してください（中のファイルだけを選択してZIP化しない）。

正しいZIP構造：
```
kiwa-personal-media.zip
  kiwa-personal-media/
    style.css
    theme.json
    functions.php
    templates/
    parts/
    patterns/
    assets/
    README.md
```

## 4. 有効化後に設定すること

1. **サイトタイトル・キャッチフレーズを設定**
   - 設定 → 一般 → サイトのタイトル：`Kiwa Personal Media`
   - キャッチフレーズ：`終わりから、これからの生き方を考える。`

2. **固定フロントページを設定**
   - 設定 → 表示設定 → ホームページの表示：「固定ページ」
   - ホームページ：「トップページ」（下記の推奨固定ページを先に作成）
   - 投稿ページ：「ブログ」

3. **パーマリンクを設定**
   - 設定 → パーマリンク → 投稿名（`/%postname%/`）を推奨

4. **ナビゲーションメニューを設定**
   - 外観 → サイトエディター → ナビゲーション から編集

5. **お問い合わせフォームプラグインを確認**
   - Contact Form 7 または WPForms Lite（無料）を推奨

---

## 5. トップページについての重要な注意

WordPressの固定ページ「Home」を編集しても、テーマに `front-page.html` が存在する場合は公開トップページに反映されないことがあります。公開トップページ（`http://kiwa-blg.com/wp/`）を変更する場合は、**外観 → エディター → テンプレート → フロントページ** を編集してください。

固定ページ「Home」の内容は、WordPressのエディタープレビューには表示されますが、実際の公開URLでは `front-page.html` テンプレートが優先されます。

## 6. テーマファイル更新後の反映について

テーマファイルを更新した場合、サイトエディターで以前に個別編集したテンプレートには
自動では反映されないことがあります。

**テーマ更新後の確認手順：**

1. 外観 → エディター → テンプレート から対象テンプレートを開く
2. 右上メニュー（⋮）→「テンプレートをリセット」を選択
3. 固定ページ・投稿の本文は消えません（データベース側に保存済み）

**特にリセット確認が必要なテンプレート：**

- フロントページ（Front Page）— トップページのレイアウト変更時
- 投稿ページ（Single Post）— 記事テンプレート変更時
- ホーム（Home / ブログ一覧）— ブログ一覧テンプレート変更時

**注意：**
テンプレートのリセットは「カスタマイズ済みの編集内容」を消しますが、
投稿・固定ページの本文やメタ情報はデータベースに保存されているため消えません。
リセットが心配な場合は、リセット前にサイトエディターでスクリーンショットを撮っておいてください。

## 7. サイトエディターで編集する場所

外観 → サイトエディター から以下を編集できます。

| 編集したいもの | 場所 |
|---|---|
| ヘッダー（ロゴ・メニュー） | テンプレートパーツ → Header |
| フッター（説明・SNSリンク） | テンプレートパーツ → Footer |
| トップページ全体 | テンプレート → Front Page |
| ブログ一覧 | テンプレート → Home |
| 投稿詳細ページ | テンプレート → Single |
| 固定ページ | テンプレート → Page |
| Hero / About / Topics等 | パターン → Kiwa Sections |

## 8. 推奨固定ページ

以下の固定ページを作成してください。WordPressが `/wp/` サブディレクトリの場合、
URLは `/wp/about/` のようになります。

| ページ名 | スラッグ | URL例（/wp/配下の場合） |
|---|---|---|
| トップページ | `top` または `home` | /wp/ |
| About（さとるんについて） | `about` | /wp/about/ |
| Blog | `blog` | /wp/blog/ |
| Works（制作・実験） | `works` | /wp/works/ |
| Contact（お問い合わせ） | `contact` | /wp/contact/ |
| Privacy Policy | `privacy-policy` | /wp/privacy-policy/ |

---

## 9. 推奨メニュー

以下のリンクをナビゲーションメニューに設定してください。
※ WordPressが `/wp/` サブディレクトリにある場合は `/wp/` を付けてください。

- **Home** → `/wp/`
- **About** → `/wp/about/`
- **Blog** → `/wp/blog/`
- **Works** → `/wp/works/`
- **Contact** → `/wp/contact/`

## 12. 勤務先や個人が特定されないための運用ルール

### 書いてよいこと
- 「人生の終わりに関わる仕事」という表現（業界の直接名称は避ける）
- 「仕事を通じて考えたこと」という一般的な言い回し
- 個人の経験・考察・感情（具体的な人物・場所・案件名は伏せる）
- AIやWebツールの操作・活用記録（実務案件の詳細は伏せる）

### 書いてはいけないこと
- 会社名・勤務先名・施設名・所在地
- 実在する特定の人物（パートナー・同僚・顧客等）の詳細
- 具体的な案件名・クライアント情報
- 遺族・故人に関わる情報
- 特定の地域名・施設名
- 実在の葬儀社・宗教施設・病院名
- 交際年数・交際相手の職業・外見的特徴など特定につながる情報

### 画像について
- 顔写真は投稿しない
- 職場・特定の場所が映り込む写真は投稿しない
- `assets/images/` フォルダに将来追加する場合も同ルールを守る

### SNSとの連携について
- 実名SNSアカウントとは別の匿名アカウントで運用することを推奨
- サイトURL・SNSアカウントのリンクは意図して公開するもののみ記載
