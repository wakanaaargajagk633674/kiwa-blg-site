Prompt、Skill、Agentは、賢さの順番ではなく運用単位の違いとして見る。

## Prompt

Promptは、入力、制約、出力形式、判断基準を揃えるための作業指示書。

## Skill

Skillは、繰り返す作業を再利用できるように保存した手順。検証方法と禁止事項まで含めると、品質が安定する。

## Agent

Agentは、目的、権限、ツール、完了条件を持って実行するAIスタッフ。任せる前に、どこまで自律してよいかを決める。

小さなPromptを整え、安定したものをSkill化し、実行範囲が明確なものだけAgentに渡す。
